приложение: Ascii-art
Запускается из ascii_art.py.
Принимает на вход (jpg, png, jpeg, mp4, avi, webm)-файл и выдает ascii-art.txt или ascii-art.avi

Применение: После запуска, выбрать файл в формате jpg, png, jpeg, mp4, avi, webm
установить кол-во символов в длину и высоту, после чего его можно конвертировать
длина-высота не может превышать размеры оригинала
рекомендованые длина/высота: 150/100, 250/200, 200/150, 100/75
есть возможность изменить контрастность изображения/видео и выбрать необходимый момент в видео для предпросмотра.
доступен предпросмотр кадра с измененной контрастностю в оригинале, черно-белом варианте
